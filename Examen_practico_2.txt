LABORATORIO 3 (Backups Remotos) -->

	Crear claves Privadas y Publicas : 

		$ssh-keygen -t rsa
		$/home/lsi/id_rsa
		$ssh-add /home/lsi/.ssh/id_rsa

	Abrir una conexion ssh desde la terminal : 
		$ssh usuario@IP_EXTERNA --> Se introduce Yes
		$exit (Para cerrar la conexion)

	Backup Remoto con rsync  : 

		$ssh lsi@IP_EXTERNA
		$mkdir nombre_carpeta
		$rsync -arv -e ssh /home/usuarioPC/Seguridad lsi@IP_EXTERNA:/home/lsi/nombre_carpeta

	Backup Remoto con cp : 

		$ssh lsi@IP_EXTERNA
		$mkdir nombre_carpeta
		$scp /home/usuarioPC/Seguridad/nombre_archivo_a_copiar lsi@IP_EXTERNA:/home/lsi/nombre_carpeta
		*(El scp hay que hacerlo con todos los archivos a enviar no se puede copiar una carpeta)*

LABORATORIO 5 (gpg) -->

	Web : https://elbauldelprogramador.com/chuleta-de-comandos-para-gpg/

	#Cifrar Archivo#
	#En caso de tener que instalar gpg --> sudo aptget-install gnupg2#
	1. Se crea el archivo a cifrar (Ej : archivo.txt)
	2. gpg -c archivo.txt
	3. Introducir una clave para cifrarlo

	#Descifrar Archivos#
	1. gpg privado.txt.gpg
	2. Introducir la clave con la que se ha cifrado
	3. Si se descifra, se crea un archivo .txt

	#Crear una clave#
	gpg --gen-key

	#Exportar clave publica#
	gpg --export -a "Nombre de usuario"

	#Expotar clave privada#
	gpg --export-secret-key -a "Nombre de Usuario" > private.key

	#Importar clave publica#
	gpg --import public.key

	#Importar clave privada#
	gpg --allow-secret-key-import --import private.key

	#Eliminar una clave publica del anillo#
	gpg --delete-key "Nombre de usuario"

	#Eliminar una clave privada del anillo#
	gpg --delete-secret-key "Nombre de usuario"

	#Listar claves publicas#
	gpg --list-keys

	#Listar claves privadas#
	gpg --list-secret-keys
